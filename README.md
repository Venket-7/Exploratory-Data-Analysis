# Exploratory Data Analysis

This repository contains Exploratory Data Analysis (EDA) performed using Python.

## 📊 Overview
The project focuses on analyzing datasets to extract meaningful insights using statistical methods and visualizations.

## 🔍 Key Features
- Data cleaning and preprocessing
- Univariate and multivariate analysis
- Data visualization
- Insight generation

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 File
- variate_analysis.ipynb

## 🚀 Objective
To understand data patterns, detect anomalies, and prepare data for further modeling.
